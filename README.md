# Weather & Time System for Rusted Warfare

## Features
- Full day-night cycle: Day → Sunset → Night → Midnight
- Randomized weather with probabilities:
  - Clear: 50%
  - Rain: 20%
  - Fog: 20%
  - Storm: 10%
- Overlay effects simulate time-of-day color changes
- Plug-and-play: just drop into Rusted Warfare `mods` folder

## Installation
1. Download `WeatherTimeMod.zip`
2. Extract `WeatherTimeMod` folder into the Rusted Warfare `mods` folder
3. Launch the game → Mods → Local Mods → Enable “Weather & Time System”
4. Start a match and enjoy!
